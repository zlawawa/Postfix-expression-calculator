## Калькулятор постфиксных выражений

### Суть проекта:
Консольное приложение для вычисления арифметических выражений, записанных в постфиксной (обратной польской) записи. Поддерживает операции сложения, вычитания, умножения и деления над положительными числами.

### Ключевой функционал:
1. Вычисление постфиксного выражения с операциями +, -, *, /;
2. Валидация входной строки;
3. Обработка ошибок (деление на ноль, некорректный ввод).


## Структура файлов
```
pr.h - заголовочный файл с классами
pr.cpp - точка входа и меню
pr_m.cpp - реализация функции вычисления
```

## Компиляция и запуск
### Компиляция:
```
g++ pr.cpp pr_m.cpp -o ./program.exe
```

### Запуск:
```
./program.exe
```
